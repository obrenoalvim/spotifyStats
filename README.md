# SpotifyStats

Este projeto tem como intuito ajudar você a explorar e entender suas estatísticas de música no Spotify. A proposta é oferecer um espaço onde você pode:

- Conectar sua conta e visualizar um painel com suas informações.
- Ver suas músicas e artistas mais ouvidos em diferentes períodos de tempo.
- Obter insights sobre seus hábitos de audição e preferências musicais.
- Gerar papéis de parede a partir das capas dos seus álbuns favoritos.

O objetivo é proporcionar uma visão clara e agradável do seu gosto musical, de forma simples e direta.
